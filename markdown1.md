### Syntax guide(basic):
- Headers:#
- Emphasis:*,_
- List:*,[0-9]
- Img:![name](path), Format: ![Alt Text](url)
- Link:http://url - automatic, [link name](url)
- Quotes:>
- Inline code:`code`

### GFM(GitHub Flavored Markdown)
- Syntax highlighting: ```
- Task Lists
- Tables
- SHA references
- Issue references within a repository
- @mentions
- Automatic linking(URL)
- Strikethrough:
- Emoji: :+1:
